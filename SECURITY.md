# Security Policy

## Supported versions

Current development branch (`main`) is supported.

## Reporting a vulnerability

Do not open public issue for potential security vulnerabilities.

Report privately using GitHub Security Advisories:

- [Private vulnerability report](https://github.com/PeterBuchHansen/navix/security/advisories/new)

Include:

- summary of issue
- impact assessment
- reproduction details
- potential fix if known

Maintainer will acknowledge and triage as quickly as possible.

## Response goals

- acknowledge report within 72 hours
- provide status updates during triage
- publish fix and disclosure notes after remediation
